# 📋 Online Feedback & Complaint Portal

Modern web-based feedback and complaint management system with real-time email validation and admin dashboard.

## ✨ Features

- 📝 Submit complaints with file attachments
- 📧 Real-time email validation (Abstract API)
- 🚫 Disposable/fake email blocking
- 🔐 Secure admin dashboard
- 📨 Gmail SMTP email notifications
- 🔍 Complaint tracking with Tracking ID
- 🔄 Status updates with email alerts
- 🌙 Dark mode support
- 📱 Fully responsive design

## 🛠️ Technologies

- PHP 8, MySQL
- PHPMailer (Gmail SMTP)
- Abstract API (Email Validation)
- HTML5, CSS3, JavaScript

## ⚙️ Setup

1. Clone repository
2. Import `database.sql` to MySQL
3. Rename `config/mail.php.sample` to `config/mail.php`
4. Add Gmail & API credentials
5. Start XAMPP
6. Visit `http://localhost/feedback-portal/`
